# Телеграм-бот ГАЗОБЛОК34 — приём заказов

Готовый бот, который принимает заявки прямо в Telegram и пересылает их вам в личку.
Клиент проходит короткий диалог (имя → телефон → что нужно → комментарий), бот
собирает заявку и отправляет её вам, а клиенту говорит «спасибо, скоро свяжемся».

На сайте кнопки **«Заказ через Telegram-бот»** уже ведут на бота
(в форме заказа и в шапке). После запуска просто укажите имя бота — см. шаг 5.

---

## Что нужно (один раз, ~10 минут)

### 1. Создать бота в Telegram
1. Откройте в Telegram чат с **@BotFather**.
2. Отправьте команду `/newbot`.
3. Введите имя бота — например, `ГАЗОБЛОК34 — заказы`.
4. Введите username (должен заканчиваться на `bot`) — например, `gazoblok34_bot`.
5. BotFather пришлёт **токен** вида `123456789:AAExxxxxxxxxxxxxxxxxxxxxxxxxxx` — скопируйте его.

### 2. Узнать свой Telegram ID (куда слать заявки)
1. Откройте чат с **@userinfobot** и нажмите Start.
2. Он пришлёт ваш `Id` — число, например `461025478`. Это `ADMIN_CHAT_ID`.

> Хотите получать заявки в группу? Добавьте бота в группу, сделайте администратором,
> и в качестве `ADMIN_CHAT_ID` используйте ID группы (он начинается со знака `-`).

### 3. Установить Python и библиотеку
На компьютере или сервере с установленным Python 3.10+:
```bash
pip install "python-telegram-bot==21.6"
```

### 4. Прописать токен и ID
Откройте `bot.py` и впишите свои значения вверху файла:
```python
BOT_TOKEN = "сюда токен от BotFather"
ADMIN_CHAT_ID = 461025478   # ваш Telegram ID числом
```
Либо задайте их через переменные окружения (безопаснее):
```bash
export GB_BOT_TOKEN="123456789:AAE..."
export GB_ADMIN_CHAT_ID="461025478"
```

### 5. Прописать бота на сайте
В админке сайта (`admin.html`) → вкладка **Контакты** замените ссылку, либо
в файле `app/data.js` найдите блок `TG_BOT` и впишите username вашего бота:
```js
TG_BOT: {
  username: "gazoblok34_bot",
  link: "https://t.me/gazoblok34_bot",
},
```

### 6. Запустить
```bash
python bot.py
```
В консоли появится «Бот запущен». Откройте бота в Telegram, нажмите **Start** —
и проверьте, что заявка приходит вам в личку.

---

## Чтобы бот работал круглосуточно
Бот должен работать на постоянно включённом устройстве. Варианты:
- **VPS** (от ~200 ₽/мес): timeweb, reg.ru, beget. Запустить как сервис systemd.
- **Бесплатно для теста:** оставить запущенным на своём компьютере (бот работает,
  пока открыт терминал).

Пример автозапуска через systemd — в конце `bot.py` (в комментарии).

## Что бот умеет
- `/start` — приветствие и кнопка «Оставить заявку».
- Пошаговый сбор заявки: имя → телефон → что нужно → комментарий.
- Кнопка «Отправить контакт» — клиент делится номером в один тап.
- Пересылает готовую заявку вам с именем, телефоном и запросом.
- `/cancel` — отменить заполнение.

Дорабатывается легко: тексты и пункты меню — вверху `bot.py`.
